# ceo-bowling
## Instructions 

### Prerequisites
- Node and npm installed

### Step 1
Run
```
npm i 
```
in the root directory.

### Step 2
Run one of these commands to run the respective part of the test

- npm run dev-part-1
- npm run dev-part-2
- npm run dev-part-3
- npm run dev-part-4
- npm run dev-part-5
